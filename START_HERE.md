## 🎯 Быстрый старт (5 шагов)

### 1️⃣ Подготовьте данные (5 мин)

```bash
# 1. Получите токен бота от @BotFather
@BotFather → /newbot → сохраните токен

# 2. Сгенерируйте секретный ключ
python3 -c "import secrets; print(secrets.token_urlsafe(32))"

# 3. Подготовьте данные Marzban
URL: https://your-marzban.com
Username: admin
Password: ваш_пароль
```

### 2️⃣ Создайте проект на Railway (3 мин)

1. Откройте [railway.app](https://railway.app)
2. "Deploy from GitHub repo"
3. Выберите ваш репозиторий

### 3️⃣ Добавьте сервисы (5 мин)

```
+ New → Database → Redis
+ New → Empty Service → "telegram-bot"
+ New → Empty Service → "api"
+ New → Empty Service → "webapp"
```

### 4️⃣ Настройте переменные (10 мин)

Для каждого сервиса скопируйте переменные из **DEPLOYMENT_CHECKLIST.md**

### 5️⃣ Запустите! (2 мин)

- Сгенерируйте публичные URL для API и WebApp
- Обновите CORS в API
- Настройте Menu Button в BotFather

**Готово! ✨**
